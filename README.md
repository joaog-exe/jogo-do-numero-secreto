<h1> 🎮 Jogo do Número Secreto </h1>

Este projeto foi desenvolvido como parte do curso de **Lógica de Programação** da [Alura](https://www.alura.com.br/), desenvolvido em **JavaScript**, onde o usuário deve adivinhar um número aleatório gerado pelo sistema. O projeto utiliza manipulação do DOM e a API de síntese de voz para tornar a experiência mais acessível e dinâmica.

<h1> 🚀 Funcionalidades </h1>

* **Geração Inteligente:** O sistema gera um número entre 1 e 10, garantindo que o mesmo número não seja repetido em rodadas consecutivas até que todas as opções sejam esgotadas.
* **Feedback em Tempo Real:** O jogo informa se o número secreto é maior ou menor que o chute realizado.
* **Contagem de Tentativas:** Ao acertar, o jogo exibe o número de tentativas que o usuário levou para vencer.
* **Acessibilidade (Web Speech API):** O jogo utiliza narração de voz (`speechSynthesis`) para ler as mensagens exibidas na tela em português (pt-BR).
* **Reinicialização:** O sistema gerencia o estado do botão "Novo Jogo", habilitando-o apenas quando a partida é concluída.
